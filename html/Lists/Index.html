Unordered Lists

Unordered lists are used to list out things that doesn't need to be in an specific order. Example: A shopping list.

A shopping list

  • Chocolates
  • Cabbages
  • Mineral water
  • Eggs

Ordered lists

Ordered Lists are used to list out things that need to be in a specific order. Example: Stepwise Manual.

A manual to open door

  1. Go to a door
  2. Twist the doorknob
  3. Pull the door
